CMAuthProvider Protocol Reference

Conforms to NSObject
Declared in CMAuthProvider.h

Overview

CMAuthProvider is a simple protocol that can be easily implemented to include just about any type of request authorization. Cumulus includes built in implementations of CMAuthProvider that provide BASIC, CMServerTrustAuthProvider, CMClientCertificateAuthProvider, CMOAuth2AuthProvider and CMS3AuthProvider authoriization.
